# My Claude Code Skills

这个仓库存储我所有的Claude Code skills，通过自动同步工具维护。

## Skills列表

- openspec-cn: 技术方案规划工具（中文版）
- general-project-workflow: 通用项目开发流程
- miniprogram-workflow: 小程序专用开发流程（含腾讯云官方skills）
- project-workflow: 项目开发流程（旧版，已弃用）
- auto-sync-skills-to-github: Skills自动同步工具

## 同步信息

- **同步方式**: SSH (无需token，更安全)
- **自动触发**: 每次Claude Code启动时
- **手动触发**: 对话中说"同步skills到GitHub"

## 技术栈

- Git
- GitHub SSH Key
- Claude Code Skills
